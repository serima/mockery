Defines what a quick definition should produce.

protected string $_quickDefinitionsApplicationMode = self::QUICK_DEFINITIONS_MODE_DEFAULT_EXPECTATION
Possible options are: - self::QUICK_DEFINITIONS_MODE_DEFAULT_EXPECTATION: in this case quick definitions define a stub. - self::QUICK_DEFINITIONS_MODE_MOCK_AT_LEAST_ONCE: in this case quick definitions define a mock with an 'at least once' expectation.
